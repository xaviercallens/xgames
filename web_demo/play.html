Proutman is a desktop game built with Pygame. It cannot run directly in the browser. Follow the installation steps below to play on your computer!

üöÄ Quick Start

1 Clone the Repository 
git clone https://github.com/xaviercallens/xgames.git
cd xgames
2 Install Dependencies 
pip install pygame torch numpy

Optional: Install FFmpeg for video recording

sudo apt-get install ffmpeg  # Ubuntu/Debian
brew install ffmpeg          # macOS
3 Run the Game 
./play_bomberman.py

Or use the unified CLI:

./bomberman play

üéÆ Game Controls

  • WASD or Arrow Keys - Move your character
  • SPACE - Drop a trump (bomb) üí®
  • C - Drop a caca (wall) üí©
  • P - Pause game
  • R - Start/Stop recording üé¨
  • S - Save statistics üìä
  • ESC - Quit game

üìã System Requirements

Minimum Requirements

  • Python 3.8 or higher
  • Pygame 2.0+
  • 1GB RAM
  • 100MB disk space

Recommended

  • Python 3.10+
  • PyTorch (for AI opponents)
  • FFmpeg (for video recording)
  • 2GB RAM
