🌟 The Story Behind Proutman

"Papa, let's make a funny game with prouts (farts) and caca (poop)! It will make learning Python fun for kids!"
— My Son's Brilliant Idea 💡

This game was inspired by my son's creativity to make programming education hilarious and engaging for children! Instead of boring examples, kids learn Python, AI, and game development by creating smelly trump bombs 💨 and strategic poop blocks 💩!

🐍

Learn Python

Real programming with real code! Kids learn variables, functions, classes, and object-oriented programming through game development.

🤖

Train AI

Kids can train their own AI opponent using reinforcement learning! Watch the AI get smarter in real-time.

🎮

Build Games

Learn game development from scratch! Graphics, sound, controls, collisions, and game states.

😂

Have Fun!

Hilarious theme with farts and poop! Kids actually WANT to learn because it's so much fun!

🎨

Beautiful Graphics

Professional sprites and animations! Learn about art, design, and visual programming.

📊

Track Progress

Statistics, achievements, and AI training metrics! Kids see their improvement over time.

🌱

Level 1: Beginner

Ages 8-10

Level 2: Intermediate

Ages 10-12

Time: 5-10 hours

🌳

Level 3: Advanced

Ages 12-14

Time: 20+ hours

Level 4: Expert

Ages 14+

Time: 50+ hours
